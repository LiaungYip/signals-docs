# About

This is a copy of the in-game wiki documentation for MineMaarten's "Signals" mod, reformatted for the web.

I've made this copy so the documentation can be read without the In Game Wiki mod installed, as IGW isn't available for Minecraft 1.10.

In case of error with this web copy, contact minecraft@penwatch.net.

For all other issues, go upstream to [https://github.com/MineMaarten/Signals](https://github.com/MineMaarten/Signals).
